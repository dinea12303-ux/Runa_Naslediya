СТАРТ

1. Открой новый репозиторий GitHub.
2. Залей в корень репозитория все файлы и папки из этого комплекта:
   - index.html
   - admin.html
   - data/books.json
   - assets/style.css
   - assets/app.js
   - assets/admin.js
   - папку books можно оставить пустой

3. В настройках репозитория включи GitHub Pages:
   Settings → Pages → Deploy from a branch → main → /root → Save.

4. Открой сайт:
   https://ТВОЙ_ЛОГИН.github.io/ТВОЙ_РЕПОЗИТОРИЙ/

5. Открой админку:
   https://ТВОЙ_ЛОГИН.github.io/ТВОЙ_РЕПОЗИТОРИЙ/admin.html

6. Во вкладке «Подключение» введи:
   - владелец репозитория: твой логин GitHub
   - название репозитория: имя нового репозитория
   - ветка: main
   - GitHub Token

7. Нажми «Сохранить настройки», потом «Проверить подключение».

8. После этого можно создавать книги и главы прямо с сайта.

ВАЖНО

Токен не записывается в файлы сайта. Он хранится только в браузере.
Картинки лучше вставлять ссылками: обложки, иконки, иллюстрации главы.
Для картинки внутри текста главы используй:

[img]https://site.com/image.jpg[/img]
